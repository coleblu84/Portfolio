## ✨ Features

- Interactive flip-card UI built with JavaScript
- Smooth card animations and transitions
- Social media integration (LinkedIn, GitHub, Email)
- Responsive layout using CSS Grid and Flexbox
- Hover effects for improved interactivity
- External links open in new tabs for better UX

## ⚡ Updates & Improvements

This project is a redesigned version of an earlier portfolio. Key improvements include:

- Redesigned layout for a more modern and structured UI
- Improved responsiveness across different screen sizes
- Refactored card system using JavaScript for interactivity
- Enhanced spacing, typography, and visual hierarchy
- Cleaner and more maintainable CSS architecture 

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation & event handling)
- Responsive design principles
